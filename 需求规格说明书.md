# 需求规格说明书

## 1. 项目背景

### 1.1 项目概述
LuxuryStay 精品酒店预订平台是一个基于 Spring Boot 3.2.x 的单体应用，使用 Thymeleaf 模板引擎进行服务端渲染，旨在为用户提供优雅、便捷的酒店预订体验，支持普通用户预订和酒店商家管理。

### 1.2 技术架构
- **后端框架**：Spring Boot 3.2.x + Spring Data JPA + Spring Security
- **视图模板**：Thymeleaf
- **数据库**：SQLite
- **安全认证**：JWT Token
- **前端样式**：Bootstrap 5.3.x + CSS + JavaScript
- **架构模式**：单体应用，服务端渲染
- **部署方式**：Java 应用直接部署

### 1.3 目标用户画像
- **高端商务人士**：追求品质住宿，注重服务细节
- **度假旅游者**：寻找特色精品酒店，追求独特体验
- **年轻白领**：有一定消费能力，注重生活品质
- **家庭出游**：需要舒适、安全的住宿环境

### 1.3 项目目标
- 打造高端、优雅的品牌形象
- 提供流畅、直观的预订流程
- 实现响应式设计，支持多设备访问
- 优化性能，确保快速加载

### 2.2 后端 API 需求

#### 2.2.1 酒店相关 API

**获取所有酒店列表**
- **接口**：`GET /api/hotels`
- **描述**：获取所有酒店信息
- **请求参数**：无
- **响应**：`List<HotelDTO>`
- **字段**：
  - id: 酒店ID
  - name: 酒店名称
  - location: 所在城市
  - description: 酒店描述
  - rating: 评分
  - price: 价格
  - imageUrl: 图片URL
  - tags: 标签列表

**获取精选酒店**
- **接口**：`GET /api/hotels/featured`
- **描述**：获取精选酒店列表（前6个）
- **请求参数**：无
- **响应**：`List<HotelDTO>`

**获取城市列表**
- **接口**：`GET /api/cities`
- **描述**：获取推荐城市列表
- **请求参数**：无
- **响应**：`List<CityDTO>`
- **字段**：
  - id: 城市ID
  - name: 城市名称
  - hotelCount: 酒店数量
  - startPrice: 起始价格
  - imageUrl: 城市图片URL

#### 2.2.2 评价相关 API

**获取用户评价**
- **接口**：`GET /api/reviews`
- **描述**：获取最新用户评价（前4条）
- **请求参数**：无
- **响应**：`List<ReviewDTO>`
- **字段**：
  - id: 评价ID
  - userName: 用户姓名
  - avatarUrl: 头像URL
  - content: 评价内容
  - rating: 评分
  - hotelName: 酒店名称
  - date: 评价日期

#### 2.2.3 服务相关 API

**获取特色服务**
- **接口**：`GET /api/services`
- **描述**：获取所有特色服务
- **请求参数**：无
- **响应**：`List<ServiceDTO>`
- **字段**：
  - id: 服务ID
  - icon: 图标名称
  - title: 服务标题
  - description: 服务描述

#### 2.2.4 API 规范

**响应格式**
```json
{
  "code": 200,
  "message": "success",
  "data": {}
}
```

**错误码**
- 200: 成功
- 400: 请求参数错误
- 404: 资源不存在
- 500: 服务器内部错误

**跨域配置**
- 允许来源：`http://localhost:3000`
- 允许方法：GET, POST, PUT, DELETE, OPTIONS
- 允许请求头：*

### 3.1 性能需求
- 首屏加载时间 < 2 秒
- Lighthouse 性能分数 > 90
- 图片懒加载
- 代码分割，按需加载

### 3.2 兼容性需求
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- iOS Safari 14+
- Android Chrome 90+

### 3.3 响应式断点
- 桌面：>= 1200px
- 平板：768px - 1199px
- 手机：< 768px

- HTTPS 支持
- XSS 防护
- CSRF 防护
